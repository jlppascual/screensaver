## Development

Fork this repository and add you changes to your own fork.
Then send a pull request to this repo.

### Windows

1. Download [NodeJs v18](https://nodejs.org/en/download)

## Running

1. Download libraries.

```
npm install
```

2. Start develop view.

```
npm run dev
```
